# Continue

**Keep doing what you're doing, but take this opportunity to really reflect on what you're doing.**

Pauses to deeply reflect on current work, approach, and progress - then continues with improved awareness and potentially adjusted direction. Combines reflection with continuation.

**Use when:** You're in the middle of work and want to pause, reflect deeply, understand what you're doing and why, then continue with better awareness.

---

## Purpose

This command provides:
- **Deep Reflection**: Meta-cognitive analysis of current work
- **Approach Evaluation**: Is the current approach working? Why?
- **Pattern Recognition**: What patterns are emerging?
- **Course Adjustment**: Should we adjust direction?
- **Continued Work**: Then continue with improved awareness

---

## Philosophy

1. **Reflect While Working**: Don't stop, but pause to understand
2. **Meta-Cognition**: Think about thinking, work about work
3. **Course Correction**: Adjust if needed, but keep moving
4. **Awareness**: Continue with better understanding
5. **Iterative Improvement**: Each reflection makes work better

---

## What Gets Reflected On

### 1. Current Work State
- **What's Being Worked On**: Files, features, tasks
- **Progress Made**: What's been accomplished
- **Current Approach**: How you're working
- **Active Context**: What's in focus right now

### 2. Work Approach
- **Methodology**: Is the approach effective?
- **Efficiency**: Are you working efficiently?
- **Quality**: Is the quality good?
- **Direction**: Are you heading the right way?

### 3. Patterns & Insights
- **Emerging Patterns**: What patterns are developing?
- **Repeated Issues**: Are there recurring problems?
- **Success Factors**: What's working well?
- **Blockers**: What's slowing things down?

### 4. Reflection Questions
- **What am I doing?**: Clear understanding of current work
- **Why am I doing it?**: Purpose and rationale
- **How am I doing it?**: Approach and methodology
- **Is it working?**: Effectiveness assessment
- **Should I adjust?**: Course correction needed?

### 5. Adjusted Continuation
- **Insights Applied**: Continue with new awareness
- **Course Adjusted**: Modified direction if needed
- **Improved Approach**: Better methodology
- **Clear Next Steps**: What to do next with awareness

---

## Execution Steps

1. **Capture Current State**
   - Get current git status
   - Identify files being worked on
   - Check recent activity
   - Understand current context

2. **Deep Reflection**
   - Analyze what's being done
   - Evaluate the approach
   - Identify patterns
   - Recognize issues or opportunities

3. **Meta-Cognitive Analysis**
   - Think about the thinking
   - Evaluate the evaluation
   - Understand the understanding
   - Reflect on the reflection

4. **Insight Generation**
   - What have we learned?
   - What should we adjust?
   - What should we continue?
   - What should we change?

5. **Adjusted Continuation**
   - Apply insights
   - Adjust course if needed
   - Continue with awareness
   - Clear next steps

## Integration with Other Commands

- **`/checkpoint`**: Creates checkpoint (different from continue)
- **`/consider`**: Analyzes options (continue reflects on current)
- **`/resume`**: Picks up from last session (continue reflects on current)
- **`/analyze`**: Analyzes Phase 1 data (continue reflects on current work)
- **`/phase1`**: Gets full overview (continue is more focused)
