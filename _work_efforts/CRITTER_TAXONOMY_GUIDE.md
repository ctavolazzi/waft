## Overview

The WAFT system uses a sophisticated taxonomic classification system to name and categorize digital organisms (critters). Each organism receives a unique scientific name based on its genetic code (genome ID), creating a deterministic and scientifically rigorous naming system.

---

## Scientific Naming Format

All digital organisms are named using the following format:

**"Genus Species, Title"**

### Examples:
- **Cognis Novus, the Fragile** (Latin culture)
- **Prana Adi, the Swift** (Sanskrit culture)
- **Fenris Fyrsti, the Great** (Old Norse culture)
- **Aura Alpha, the Bold** (Cyber/Tech culture)

### Components:

1. **Genus** (Capitalized): The primary taxonomic category, indicating the organism's cultural lineage
2. **Species** (Capitalized): The specific identifier within the genus
3. **Title** (Lowercase with "the"): A descriptive epithet that adds character to the organism

## How Names Are Generated

### Deterministic Process

1. **Genome ID**: Each organism has a unique SHA-256 hash (64 hexadecimal characters) that represents its genetic code
   - Includes: agent configuration, code hash, state schema version

2. **Culture Selection**: First byte (first 2 hex characters) determines naming culture
   - 0x00-0x3F → Sanskrit
   - 0x40-0x7F → Old Norse
   - 0x80-0xBF → Latin
   - 0xC0-0xFF → Cyber/Tech

3. **Genus Selection**: Bytes 2-5 (hex characters 4-10) select from culture-specific genera list

4. **Species Selection**: Bytes 6-8 (hex characters 10-16) select from culture-specific species list

5. **Title Selection**: Last 4 bytes (hex characters 56-64) select from universal titles list

6. **Archetype Selection**: Bytes 4-5 (hex characters 8-12) determine anatomical archetype

### Key Properties:

- **Deterministic**: Same genome ID always produces the same name
- **Unique**: Different genome IDs produce different names (with very high probability)
- **Scientific**: Follows Linnaean taxonomy principles
- **Multilingual**: Four distinct cultural lineages
- **Capitalized**: All names properly formatted

## Hybrid Names

When organisms reproduce, they can create hybrid names that blend two parent cultures:

**Example**: **Rishi Vindr, the Eternal**
- Genus "Rishi" from Sanskrit parent
- Species "Vindr" from Old Norse parent
- Title from combined genome hash

This creates linguistic diversity and reflects the evolutionary process.

## Technical Details

### Genome ID Calculation

The genome ID is a SHA-256 hash of:
```json
{
  "config": {
    "role": "...",
    "goal": "...",
    "backstory": "...",
    "tools": [...]
  },
  "code_hash": "SHA-256 of agent class source code",
  "state_version": "schema version number"
}
```

**Critical**: Uses `json.dumps(..., sort_keys=True)` for deterministic hashing.

### Name Generation Algorithm

```python
# Pseudo-code
genome_id = sha256(genome_data)  # 64 hex characters

first_byte = int(genome_id[0:2], 16)  # 0-255
culture = select_culture(first_byte)  # Sanskrit, Norse, Latin, or Cyber

genus_index = int(genome_id[4:10], 16) % len(culture.genera)
species_index = int(genome_id[10:16], 16) % len(culture.species)
title_index = int(genome_id[56:64], 16) % len(universal_titles)

genus = culture.genera[genus_index]
species = culture.species[species_index]
title = universal_titles[title_index]

scientific_name = f"{genus} {species}, {title}"
```
